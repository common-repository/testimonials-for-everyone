== Installation ==

ğŸš€ **Follow these steps to install and start using the plugin:**

1. Download the plugin from the WordPress plugin directory or upload the `testimonials-for-everyone` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create a new page or post, click the "+" button in the Gutenberg editor to add a block, and search for "Testimonials for Everyone".
4. Customize the testimonial block using the available settings (layout, colors, borders, sorting options, etc.).
5. Publish the page or post, and the testimonials will be live on your site!
